Project Name - Finance Manager android mobile app.
Project Purpose - To build an android mobile app for MSEs, to assist with managing the finances effectively in their business.
Problems to solve - Businesses are less effective in money mangement in terms of budgeting, utilization and lean practices.
Solutions to be given - A system that is less complex and helpful for managing data offline and making the user track his asset and manage it well.
Language and tools used - Kotlin, Jetpack compose, Android studio (IDE).
Project status - Started and in progress
